วิธีคำนวณ SMA & EMA
SMA (Simple Moving Average)

ค่าเฉลี่ยธรรมดา — ทุกวันมีน้ำหนักเท่ากัน

SMA(n) = (P₁ + P₂ + ... + Pₙ) / n

เช่น SMA3 = (ราคาวันที่1 + วันที่2 + วันที่3) / 3

EMA (Exponential Moving Average)

ค่าเฉลี่ยถ่วงน้ำหนัก — วันล่าสุดมีน้ำหนักมากกว่า

EMA_today = Price × k + EMA_yesterday × (1-k)
k = 2 / (n + 1)

ตัวอย่าง EMA3: k = 2/(3+1) = 0.5

  • วันล่าสุด: น้ำหนัก ~50%
  • วันก่อนหน้า: ~25%
  • 2 วันก่อน: ~12.5%
RMSE (Root Mean Square Error)

ค่าความคลาดเคลื่อน — ยิ่งต่ำยิ่งแม่นยำ

RMSE = √( Σ(actual - predicted)² / n )
